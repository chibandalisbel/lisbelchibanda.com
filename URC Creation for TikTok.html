Hosting your website can be done quickly and easily on platforms like GitHub Pages or Netlify. I'll guide you through both methods so you can choose what works best for you!

1. Hosting with GitHub Pages

GitHub Pages is a free service provided by GitHub to host static websites. Here’s how you can use it:

Step-by-Step for GitHub Pages:

  1. Create a GitHub Account (if you don’t have one already):

    • Go to GitHub and sign up.

  2. Create a New Repository:

    • Once logged in, click on the + button at the top right and select New repository.

    • Give your repository a name (e.g., lisbelchibanda.com).

    • Make sure the repository is set to Public and click Create repository.

  3. Upload Your Website Files:

    • On the repository page, click the Add file button, then choose Upload files.

    • Drag and drop your index.html and styles.css files here, and click Commit changes.

  4. Enable GitHub Pages:

    • On the repository page, go to Settings (on the top menu of the repo).

    • Scroll down to the GitHub Pages section.

    • Under Source, choose main branch and click Save.

    • GitHub will provide a URL where your site is hosted. It should be something like: https://username.github.io/repository-name/.

Example:

  • If your GitHub username is LisbelChibanda and your repository name is lisbelchibanda.com, your URL will look like:
    https://LisbelChibanda.github.io/lisbelchibanda.com/

2. Hosting with Netlify

Netlify is another easy and free option to host your site. It’s also fast and has automatic continuous deployment from GitHub if you want to integrate that in the future.

Step-by-Step for Netlify:

  1. Create a Netlify Account:

    • Go to Netlify and sign up with your GitHub or email.

  2. Deploy Your Site:

    • After signing in, click New site from Git.

    • If you are using GitHub, connect your account to Netlify.

    • Choose your repository (where you uploaded the index.html and styles.css files).

    Alternatively, if you don’t want to use GitHub, you can also drag and drop your files directly:

    • On the Netlify dashboard, click Sites.

    • Choose Add new site and then Deploy manually.

    • Simply drag and drop the folder with your index.html and styles.css files.

  3. Access Your Website:

    • Once deployed, Netlify will automatically provide a URL like https://your-site-name.netlify.app.

    • You can change the site’s name to a custom domain (like lisbelchibanda.com) if you have one.

Custom Domain (Optional):

  • If you want a custom domain like lisbelchibanda.com, you’ll need to purchase it through a provider (e.g., Namecheap, GoDaddy) and follow the steps in Netlify’s DNS settings to link it.
